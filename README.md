# music21-mscore
music21 + musescore

## References

* https://music21.org/music21docs/
* https://musescore.org/fr

## Requirements

* python 3.11

## Installation

```shell
# venv
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install music21 --break-system-packages

# libb2
brew install libb2

# configure music21
python3 -m music21.configure
```
